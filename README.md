# Jeff Brown Yachts — Home V3.31

Luxury brokerage homepage with cinematic scroll storytelling.

## View live
Open `index.html` in a browser, or visit the deployed GitHub Pages URL.

## Structure
- `index.html` — main homepage (self-contained, embedded fonts)
- `JBY-V3.3-assets/` — images, videos, logos, fonts

## Features
- 3 autoplay hero videos (Axopar → Riva → Brabus) with 5s auto-cycle
- Sticky navy gradient header + section index navigation
- Featured event with scroll-driven zoom to full screen
- Three Decades — scroll-driven navy color transition (Porsche-style)
- Builders slider with 8 brands + Riva video hover
- Vessels grid with buttery Porsche-style column expansion
- Owners video expanding to full viewport
- Leaflet map with 9 office pins
